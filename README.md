# Enterprise Network Penetration Testing Lab

## Overview
This project demonstrates a penetration testing engagement conducted against a simulated enterprise environment as part of a cybersecurity course.

The objective was to identify vulnerabilities within the network and web application infrastructure, exploit discovered weaknesses, and evaluate the overall security posture.

## Methodology

The assessment followed structured penetration testing phases:

1. Reconnaissance
2. Enumeration
3. Vulnerability Discovery
4. Exploitation
5. Privilege Escalation
6. Internal Pivoting
7. Post-Exploitation Analysis
8. Remediation Recommendations

The methodology aligns with industry standards including **NIST SP 800-115** and **PTES (Penetration Testing Execution Standard).**

## Tools Used

- Nmap
- Burp Suite
- Hydra
- SQLMap
- John the Ripper
- Proxychains
- LinPEAS
- Netcat

## Key Findings

- Weak authentication mechanisms
- SQL injection vulnerability
- Local file inclusion
- Privilege escalation opportunities
- Credential reuse across systems

## Skills Demonstrated

- Network Reconnaissance
- Web Application Security Testing
- Vulnerability Assessment
- Privilege Escalation Analysis
- Security Risk Evaluation
- Security Reporting
